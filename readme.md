# Оренда: депозит і копілка (GitHub Pages)

Сайт для старого смартфона: великі плитки з цифрами.
Показує тільки **лютий** (зверху) і **січень** (знизу).

## Де змінювати цифри
В репозиторії відкрий `data.json` → ✏️ Edit → зміни числа → Commit changes.

Файл `data.json` містить:
- `deposit`: сума депозиту
- `piggy`: сума “копілки”
- `months.jan` та `months.feb`: цифри для місяців

Поля місяця:
- `d`: значення для “Д.”
- `vv`: значення для “В.В.”
- `elektro`: значення електро (може бути від’ємним), показується як `( -820 електро )`
- `sumarno` НЕ вводиться — сайт рахує: `d + vv + elektro`

## Як увімкнути GitHub Pages
Settings → Pages → Deploy from a branch → main / (root) → Save
